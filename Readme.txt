
Study Bot using Discord.py
Description
This Python script utilizes the Discord.py library to create a study bot for Discord servers. T
he study bot helps users create and manage study sessions,
set reminders, and collaborate with fellow learners. Users can join study sessions, receive reminders, and track their study progres

Required

bot_token: A unique identifier representing the study bot. This string is created by Discord API.
channel_id: An integet representing the ID of the channel the bot will respond in.


Run pip install Discord.py
pip install pytz (time formatting)
Version 2.32
